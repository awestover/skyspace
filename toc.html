Topic Description Intensity
Fundamental Theorem of Calculus introduce basic notions in calculus, prove fundamental theorem of calculus (incredibly rigorously with a ton of analysis) Technically no preqeqs, but it’ll be a bit cooler if you know calculus
Digital Signal Processing what are signals? Fourier transform, convolutions, LTIs. should know some linear algebra
Neat Problems some interesting problems / riddles I’ve heard. Especially weird stuff, and stuff that can be done while e.g. hiking. varies by the problem discussed
Funtional Analysis Build up to, and proof of Projection Theorem. Might also talk about general Banach spaces but probably mostly Hilbert spaces. Technically no prereqs, but it’ll be easier to follow if you’ve had an introductory college course in Linear Algebra and Real Analysis ;)
