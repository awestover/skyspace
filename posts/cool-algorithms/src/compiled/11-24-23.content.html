Problems (from “Parameterized Algorithms” textbook):

Does a graph \(G\) contain \(k\) vertex-disjoint triangles?

Does a graph \(G\) contain a subgraph isomorphic to a \(k\)-vertex tree \(T\)?

At the moment this blog post is a bit of a failure because the bound claimed in the textbook is that we can give a \(2^{\mathcal{O}(k)}n^{\mathcal{O}(1)}\) algorithm for both of these problems but my algorithms are more like \(k^{k}n^{\mathcal{O}(1)}\) flavor. Hopefully I’ll fix this soon! If you have ideas lmk.

Theorem. There is a \(k^{k}n^{\mathcal{O}(1)}\) algorithm for this problem.

Proof. Assume that \(G\) is a yes-instance.

\(k\)-color the vertices. Hope that your triangles get highlighted. They do with probability \(\frac{1}{k^{2k}}\). If they get highlighted they’re easy to find.

Theorem. \(k!n^{\mathcal{O}(1)}\) algorithm for iso tree

Proof. Number the tree. Color the edges. Hope they got colored in the same way we numbered them. Do a DP: “can I make it so that this vertex is the right place to attach the next edge?”
